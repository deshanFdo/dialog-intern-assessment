Dialog Axiata is Sri Lanka's flagship telecommunications service provider, known for driving innovation and digital transformation across the nation. The company's core values encompass Uncompromising Integrity, Service Excellence, and Teamwork, which are the foundational pillars of its corporate culture. Through continuous investment in advanced technologies like 5G, IoT, and AI, Dialog strives to empower and enrich Sri Lankan lives and enterprises.

For this intern assessment, we have developed a state-of-the-art Retrieval-Augmented Generation (RAG) system designed to answer user questions using strict context grounding. By ingesting technical documentation, company guidelines, or standard operating procedures, this application allows users to query a robust AI assistant that is guaranteed not to hallucinate outside the provided materials.

The architectural foundation of this system relies on FastAPI to provide a lightning-fast backend API, combined with an interactive, dynamic frontend built using React and Vite. Document ingestion handles plain text as well as PDF files, seamlessly chunking the data and indexing it into a highly efficient BM25 retrieval store, ensuring maximum performance even on free-tier, memory-constrained cloud environments.

Finally, we integrate the powerful Google Gemini Free API for our Large Language Model interactions. By applying strict role constraints and system instructions, the model is guided to behave exclusively as a context-grounded assistant. If a user asks a question whose answer does not explicitly exist within the uploaded documents, the system is strictly programmed to reply with 'I don't know'.
